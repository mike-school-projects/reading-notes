CSS Transitions & Animations

1. What does a CSS transition allow the developer to do to an element? Transition allows us to smoothly transform from one state to another to blend in the transition.
2. How does a CSS animation differ from a CSS transition? Transition moves between two states.  Animations move between multiple states.
